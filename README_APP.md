# Beast Mode — App Instructions

Diese Datei enthält Start- und Deploy-Anweisungen für die Streamlit-App `app.py`.

Schnellstart
1. Erstelle ein virtuelles Environment (empfohlen):

```bash
python -m venv .venv
source .venv/bin/activate
```

2. Installiere Abhängigkeiten:

```bash
pip install -r requirements.txt
```

3. Starte die App lokal:

```bash
streamlit run app.py
```

Hinweis: Die Datei `app.py` enthält eine verbesserte Budget-Sortierung (numerische Erkennung von Beträgen und "k"-Suffixen). Anpassungen an den Daten kannst du direkt in `app.py` vornehmen.

Wenn du möchtest, helfe ich beim Deployment auf Streamlit Cloud oder anderen Plattformen.
